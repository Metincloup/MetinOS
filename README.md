# MetinOS

I am doing this os for learning assembly and as a hobby


### **requirement**

+ nasm
+ qemu
+ some kind of hex editor(i use *Ghex*)


## Getting Started

```shell
$ git clone https://github.com/Metincloup/MetinOS.git
$ cd MetinOS
$ ./build.sh
```
